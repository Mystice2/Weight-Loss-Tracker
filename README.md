# Japan Cut Tracker

Aggressive 5-month weight loss tracker PWA for Japan trip.

## Features
- Daily logging (weight, calories, protein, steps, workout)
- Progress dashboard with 6 key metrics
- Weight trend chart
- Weekly meal system with images
- Pre-built meal suggestions
- PWA installable on iPhone

## Quick Start
1. Open `index.html` in a browser
2. Configure your goals in Settings
3. Choose a meal from Plan tab or create your own in Meals tab
4. Log your daily progress

## Installation (iPhone)
1. Open the app in Safari
2. Tap Share button
3. Select "Add to Home Screen"
